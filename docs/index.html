Study Notes on R

Types of Variables

This is a summary of R data types. It shows how to create variable and check its type.

#integer
x <- 3L
typeof(x)
## [1] "integer"
#double
y <- 2.5
typeof(y)
## [1] "double"
#character
z <- "a"
typeof(z)
## [1] "character"
#complex
c <- 3 + 5i
typeof(c)
## [1] "complex"
#logical
b <- TRUE
typeof(b)
## [1] "logical"

Simple Arithmetic Operations
